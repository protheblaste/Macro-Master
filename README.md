# 🖱️ MacroMaster v1.0
A high-speed, modern Auto-Clicker and Macro tool for Windows and Linux.

## 🚀 Quick Start (No Python Needed!)
If you just want to use the program, go to the **[Releases](https://github.com/YOUR_USERNAME/MacroMaster/releases)** section on the right and download the `main.exe` file.

## ✨ Features
* **Custom Speed:** Set delays down to 1ms (1000 clicks per second!).
* **Global Hotkeys:** Toggle the clicker from anywhere using `Ctrl + Alt + S`.
* **Clean UI:** Simple, dark-mode interface.
* **Background Running:** Works perfectly while minimized during games or tasks.

## 🛠️ For Developers (Running from Source)
If you want to run the code manually:
1. Install requirements: `pip install customtkinter pynput`
2. Run the script: `python main.py`
